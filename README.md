😂 Random Joke Generator

A fun and lightweight web application that delivers a fresh dose of humor with every click. Perfect for a quick laugh or testing API integrations.

 🚀 Features
Instant Humour: Fetches random jokes from a public [Joke API](https://sv443.net).
Multiple Categories: Get jokes related to Programming, Dark humor, or General topics.
Typewriter Effect: Animated text display for a more engaging user experience.
One-Click Refresh: Simple "Next Joke" button to keep the laughs coming.
Clean UI: Minimalist and responsive design that works on all devices.

 🛠️ Tech Stack
HTML5 & CSS3 - For the structure and "Card" based layout.
JavaScript (ES6+) - Fetch API for real-time data retrieval.
External API: Integrated with a public REST API for dynamic content.

💻 How It Works
1. When the user clicks the "Get Joke" button, an asynchronous request is sent to the Joke API.
2. The response is parsed into JSON format.
3. The content is then dynamically injected into the DOM using JavaScript.
4. If it's a "Two-part" joke, the setup and punchline are displayed sequentially.
